## STEP Install

1. git clone https://github.com/Asepdadan/test-smm.git
2. create database dan sesuaikan dg config .env
3. composer install
4. php artisan migrate
5. php artisan db:seed

## RUN APP
1. php artisan serve 
2. akses di browser http://localhost:8000
